# Card Frame Assets

Transparent PNG rarity frame overlays belong here.

Expected final filenames:

```text
card-frame-common.png
card-frame-uncommon.png
card-frame-rare.png
card-frame-legendary.png
card-frame-mythic.png
```

Use version suffixes only for previews or candidates:

```text
card-frame-common-v1.png
card-frame-common-v2.png
card-frame-mythic-v1.png
```

Remove the version suffix from the approved production asset.
